# SCRIPTS

`transferProxyOwnershipMainnet` FUNCTION:

1. yarn install
2. uncomment the function to run in `index.js`
3. create `.env` file or export the following variables: 
   - `PROVIDER`, 
   - `INFURA_API_KEY`,
   - `FROM_ADMIN_PRIVATE_KEY`, 
   - `PROXY_CONTRACT_ADDRESS`, 
   - `TO_ADMIN_PUB_KEY`
4. yarn start
